# 6502emu
## Description
A project to design an emulator for the 6502 CPU

## Why another emulator?
For eduacation and fun

This Family of CPUs powered machines like:
 - Apple II
 - Nintendo Entertainment System (NES)
 - Commodore PET
 - BBC Micro
 - Atari 800 and 2600
 - Commodore 64
 - and many more
